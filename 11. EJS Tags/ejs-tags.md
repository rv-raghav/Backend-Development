# EJS Tags
1. <%= variable %>                          - JS Output
2. <% console.log("hello") %>               - JS Execute
3. <%- <h1>Hello</h1> %>                    - Render HTML
4. <%% %%>                                  - Show <% or %>
5. <%# This is a comment %>                 - Stop execution
6. <%- include("header.ejs") %>             - Insert another EJS file
